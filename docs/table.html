The <table> tag defines an HTML table. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

Example

# Table Heading Table Heading Table Heading
1 Table Data Table Data Table Data
2 Table Data Table Data Table Data
{% highlight html %}
... ...
... ...
{% endhighlight%}

The scope attribute specifies whether a header cell is a header for a column, row, or group of columns or rows. The scope attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Syntax: <th scope="col|row|colgroup|rowgroup">
